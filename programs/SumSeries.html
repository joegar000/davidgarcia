This code calculates the sum of the series 1 + 1/2 + 1/3 + 1/4 + ... + 1/i from i being 1 to 10. I wrote it for my Programming Fundamentals 3 course in Fall 2019.


                    /** David Garcia

                    The purpose of this program is to calculate the sum of the series 1 + 1/2 + 1/3 + ... + 1/i from i=1 to i=10. The sum of the series for each
                    different value of i will be displayed on its own line in the console. */

                    public class SumSeriesDavidGarcia {
                    	/** The main method */
                    	public static void main(String[] args) {
                        
                    		/** Loop for printing the return values of the method m when it is given the values of 1 through 10 */
                    		for (int x = 1; x <= 10; x++) {
                    			System.out.println(m(x));
                    		}
                    	}
                    
                    
                    	/** Method name: m
                    	Purpose: To calculate the sum of the series where each term goes from 1/1 to 1/2 ... to 1/i
                    	Return value: double
                    	Parameters: double i */
                    
                    	public static double m(double i) {
                        
                    		/** Base case. If i = 0 then all terms in the series have been added */
                    		if (i == 0) {
                    			return 0;
                    		}
                        
                    		/** Find the value of 1/i, then subtract 1 from i and find the value of 1/(i-1) and add it to 1/i, continue until i = 0 */
                    		else {
                    			return (1 / i) + m(i - 1);
                    		}
                    	}
                    }
